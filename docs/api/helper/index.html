Helper

AcyOrt provides a helper API that can registers custom functions, and access built-in helpers

Built-in helpers see helper for more

Register function

// register a `_test` helper
acyort.helper.register('_test', function test() {
  return `<p>${acyort.version}</p>`
})

When use the outputHTML method under util to generate a page, you can access the function directly

<!-- use it on template -->
<div>{{ _test() }}</div>
