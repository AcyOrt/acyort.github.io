AcyOrt use GitHub issue's ID as URL permalink

Type URL
posts ${post_dir}/${issue.id}.html
category ${category_dir}/${category.id}/
tag ${tag_dir}/${tag.id}/
categories ${category_dir}/
tags ${tag_dir}/
pagination page/${page.num}/. example: tags/67789889/page/2/

page

Page uses page's name for its URL. example: page [about]title, url is about/
