    概述渲染器数据存储流程辅助函数工具类

AcyOrt 提供一个 helper 方法，除了能够访问内置的辅助渲染函数外，还可以注册一些自定义的函数

内置函数可以查看 helper 说明

注册函数

// 注册一个自定义函数 `_test`
acyort.helper.register('_test', function test() {
  return `<p>${acyort.version}</p>`
})

当使用 工具/辅助类 下的 outputHTML 方法生成页面时候，可以直接访问函数

<!-- 模板页面使用 -->
<div>{{ _test() }}</div>
