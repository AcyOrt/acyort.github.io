创建

当安装好 AcyOrt，可以使用下面命令初始化一个网站

$ acyort init <folder>

# 或者
$ cd  <folder>
$ acyort init

初始化完成，网站文件夹为以下结构

.
├─ config.yml
├─ templates/
|   └─ ccc45/
├─ scripts/

config.yml

配置文件，具体查看 配置 说明

templates

模板文件夹，网站所需要的 模板

scripts

自定义插件目录，用于存放自定义 插件
