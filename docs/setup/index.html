Once AcyOrt be installed, run the following commands to initialize AcyOrt

$ acyort init <folder>

# or
$ cd  <folder>
$ acyort init

After initialized, what your project folder will look like:

.
├─ config.yml
├─ themes/
|   └─ ccc45/
├─ scripts/

config.yml

Blog configuration file. You can configure all settings here.

themes

Themes folder. AcyOrt generates a static website by combining the site contents with the theme.

scripts

Custom scripts folder. Add script files and use AcyOrt extends API to expand your website
