It is easy to extend AcyOrt without modifying core module. There are two ways of plugins in AcyOrt Also you should enable the scripts/plugins in config.yml

Script

If your plugin is simply. Put your JavaScript files in the scripts folder and AcyOrt will load them during initialization.

Plugin

You can publish the plugin to NPM. Plugins must contain at least two files index.js, package.json

You should set the name, version and main entries in package.json

{
  "name": "acyort plugin",
  "version": "0.1.0",
  "main": "index.js"
}
