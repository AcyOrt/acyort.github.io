Welcome to the AcyOrt Document.

AcyOrt is a simple static website framework. Powered by GitHub and NodeJS. You can write posts on GitHub issues and your blog can be deploy automatically.

It only takes a few minutes to set up AcyOrt blog.

Requirements

AcyOrt is based on NodeJS, and you should install Node.js first

You should install Git for use GitHub service A GitHub account is required

Install

Once NodeJS be installed, you can install AcyOrt with npm:

$ npm i acyort -g
