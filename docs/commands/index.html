Here are commands AcyOrt provides.

init

$ acyort init folder

Initializes a new blog in folder. You can also initialize blog in current folder like bellow.

$ cd folder
$ acyort init

version

$ acyort version

Show current version.

build

$ acyort build

Generate website static files.

server

$ acyort server

Start a local server. Default address: http://127.0.0.1:2222/

AcyOrt will watch theme file changes and reload webpages automatically, which is convenient to debug theme.

You can also customize port.

$ acyort server port

clean

$ acyort clean

Clean generated files.
