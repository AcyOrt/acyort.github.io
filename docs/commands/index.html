There are some commands that you can use

init

$ acyort init folder

Initializes a new blog in folder. Or you can initialize new blog in the existing folder

$ cd folder
$ acyort init

version

$ acyort version

Displays version information.

build

$ acyort build

Generates blog static files.

server

$ acyort server

Starts a local server. Default is at http://127.0.0.1:2222/

liveReload

When the server is running, AcyOrt will watch the theme files changes and reload web pages automatically so itâ€™s not necessary to manually restart the server.

Port

$ acyort server port

You can change the port of the server

clean

$ acyort clean

Cleans the generated files.
