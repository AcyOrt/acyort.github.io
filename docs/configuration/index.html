Modify site settings in config.yml

Site

Setting Description
title the title of your blog
description the description of your blog
language the language of your blog. Need themes support. default is default.
timezone the timezone of your blog. uses the setting on your computer by default. example: America/New_York, Japan, and UTC.
menu the nav menus of your blog. default is null
theme the theme of your blog. default is ccc45

URL

Setting Description Default
url the URL of your blog. add http or https before. set as http://yoursite.com/child if necessary http://acyort.com

Directory

Setting Description Default
public_dir public folder. where the static site will be generated /
tag_dir tag directory tags
category_dir category directory categories
post_dir posts directory posts

Content

Setting Description
user your GitHub username
repository your GitHub repository name. you blog contents comes from this repository issues
token your GitHub access token. token should be separated by '#'. example: 5#c31bffc137f44faf7ef
authors GitHub usernames. the white list of issues contents for your blog

Extensions

Setting Description
scripts script file name. the script should be placed in scripts folder
plugins npm module name.

Other

Setting Description Default
per_page blog pagination. set 0 to disable pagination 10
default_category default category name uncategorized
line_numbers if show code line numbers true
order post order. order by issues create time(created) or update time(updated) created
cache for development. cache github issues data false
