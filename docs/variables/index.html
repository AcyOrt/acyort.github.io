Variables can be used in theme templates.

Config

Global config variables

Variable Description
config.title blog title
config.description blog description
config.url blog url
config.user GitHub username
config.repository GitHub repository

Page

Post template variables

Variable Description Type
page.id post id Number
page.created post created date Time String
page.updated post updated date Time String
page.prev previous post id Number
page.next next post id Number
page.title post title String
page.url post url URL String
page.author post author name, avatar, url Object
page.excerpt post excerpt html String
page.raw post origin markdown data markdown String
page.content post content html String
page.category post category id, name, url Object
page.tags post tags id, name, url Array

Page template variables

Variable Description Type

Pagination template variables

Variable Description Type
page.base base url URL String
page.title current page title String
page.prev previous page url URL String
page.next next page url URL String
page.posts page posts list Array
page.current current page number Number
page.total total page number Number

Categories template variable page is an Array. Item variables

Variable Description Type
id category id Number
name category name String
url category url URL String
posts all posts id belong to current category Array

Tags template variable page is an Array. Item variables

Variable Description Type
id tag id Number
name tag name String
url tag url URL String
posts all posts id belong to current tag Array
