AcyOrt 介绍与安装

LoeiFy Posted at Mar 24, 2016 . - uncategorized -

AcyOrt 是一个基于 Node.js 的博客工具。通过 GitHub API 获取 GitHub issues 的相关内容生成静态页面，从而生成一个博客。用户只要在 issues 上写自己的博客文章，然后一个简单命令就可以发布自己的博客了。包括自定义自己域名。

支持/功能

  • 文章
  • 评论
  • 归档
  • Rss
  • 页面
  • 标签
  • 分类
  • 主题
  • 菜单

安装

安装 AcyOrt 之前确保已经安装 Node.js 以及一个 GitHub 账号

接下来只需要通过 npm 安装 AcyOrt

$ npm install acyort-cli -g

创建网站

安装 AcyOrt 完成后，执行下列命令，将会在指定文件夹中创建网站

$ acyort create <folder>
$ cd <folder>
$ npm install

生成网站的主要文件及文件夹主要是：config.yml, themes, public

config.yml

网站的配置信息

# 标题，网站地址，描述，rss 链接
title: AcyOrt
url: http://acyort.github.io
description: A Node.js blog tool powered by GitHub.
rss: rss.xml

# 主题
theme: ccc45

# 分页
per_page: 2

# 菜单信息
menu:
  - name: home
    url: /
  - name: about
    url: /about
  - name: archives
    url: /archives
  - name: tags
    url: /tags
  - name: categories
    url: /categories

# 默认分类
default_category: uncategorized

# 评论相关用户名（多说 或者 disqus）
duoshuo:
disqus: acyort

# 作者过滤，避免其他用户在 issues 上写无关内容
authors:
  - LoeiFy
  - author0
  - author1

# 网站内容来源 issues 地址
# 格式为：用户名/项目名
issue_repo: AcyOrt/acyort.github.io

# GitHub access token(Optional)
# token should be separated by whitespace

# token: sd46a6d65a asd8sdbn

themes

主题文件夹，可以自定义主题，默认主题是：ccc45

public

网站静态文件生成路径

生成网站

通过一个简单命令即可生成网站

$ acyort build

查看 public 可以看到相关网站文件，你可以通过 git 方式提交网站到 GitHub 或者直接将里面文件发布到其他地方

本地测试

AcyOrt 提供本地测试，只需运行

$ acyort server

这时候访问 http://127.0.0.1:2222 就能够看到网站
