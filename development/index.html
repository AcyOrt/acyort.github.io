Development

Fork and Install modules

$ git clone git@github.com:acyortjs/acyort.git
$ npm i

start

Start local LiveReload server, http://127.0.0.1:2222/

$ npm start

build

Generate website static files.

$ npm run build

version

Show current version.

$ npm run version

clean

Clean generated files.

$ npm run clean

test

Test cases and code lint

$ npm t # lint and coverage test
$ npm run test:unit # unit test
$ npm run test:lint # code lint
$ npm run test:cover # coverage test
