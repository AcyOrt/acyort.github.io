AcyOrt provides useful APIs for new plugins, you can get that through global variable acyort in AcyOrt environment.

fs

const { fs } = acyort
const file = '/tmp/file.txt'
fs.outputFileSync(file, 'hello!')

fs is an instance of fs-extra.

version

console.log(acyort.version)
// 2.0.12

Get current AcyOrt version

logger

const { logger } = acyort

logger.success('success')
// ✔ success

logger.info('info')
// i info

logger.error('error')
// ✖ error

Print information on the console

renderer

const { renderer } = acyort
const config = renderer.render('yaml', { path: 'path/to/config.yml' })
// { title: 'AcyOrt' ... }

swig, yaml parser. use acyort-render

config

const { config } = acyort
console.log(config)
// { title: 'acyort', post_dir: 'posts' ... }

Current blog configuration

markeder

const { markeder } = acyort
const heading = '# An h1 header'

console.log(markeder.mark(heading))
// <h1 id="anh1header">An h1 header</h1>

markeder.lineNumbers = true
const _code = '\`\`\`\n<h1>h1</h1>\n\`\`\`'

markeder.mark(_code)
/*
<div class="hljs">
<table>
  <tbody>
    <tr>
      <td class="line">
        <pre><span>1</span></pre>
      </td>
      <td class="code">
        <pre>&lt;h1&gt;h1&lt;&#x2F;h1&gt;</pre>
      </td>
    </tr>
  </tbody>
</table>
</div>
*/

markdown parser. use acyort-marked

helper

const {
  _pages,
  _posts,
  _categories,
  _tags,
  _url,
  _time,
  __,
  _n,
} = acyort.helper.methods

Helpers can be used in plugins. Visit acyort-helper for more infomation

Special

const fn = s => s.split('').join('.')

// add custom helper function
acyort.helper.addMethod('_js', fn)
console.log(helper.methods._js('ab'))   // a.b

// reset the language files
acyort.helper.resetLocales()

server

const { status, running } = acyort.server

console.log(running)    // true
console.log(status)      // { e: 'change', path: 'path/to/file.html' }

acyort.server.start([port])

LiveReload server provides running status and serve AcyOrt service. visit acyort-server for more infomation

builder

acyort.builder.addTags(['archives'])
acyort.builder.compile('archives')
acyort.builder.output('archives', 'path/to/file', { data })

Builder helps compiling custom template, render and output rendered files. Custom templates should be put in theme layout folder.
